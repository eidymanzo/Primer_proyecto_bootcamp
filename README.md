# Primer_proyecto_bootcamp

Páginas obligatorias
La aplicación debe incluir al menos 4 páginas HTML conectadas entre sí:
•	AMANDA Y EIDY: index.html → Página principal (Home)
•	EIDY: servicios.html → Servicios o Productos (debe incluir una tabla)
•	NOELLE: equipo.html → Equipo/Staff
•	NOEMÍ: sobre.html → Sobre nosotros (About us)
•	AMANDA: contacto.html → Formulario de contacto
